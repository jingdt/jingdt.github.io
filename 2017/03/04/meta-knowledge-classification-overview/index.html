元知识基础：知识分类概述

发表于 2017-03-04

前言

所谓元XXX就是关于XXX的XXX，元知识就是关于知识的知识。元知识的重要性无需多言。可以说对元知识、元认知的掌握能力，是造成人与人之间能力巨大差异的主要因素。元知识能力强的人，在学习速度、学习效果、思维质量方面都会优于常人，在时间复利的影响下，这种差距会越拉越大。对这背后的原理缺乏基本了解的人，会错误的认为别人的超强能力是一种天赋。

元知识涉及的范畴很大，在哲学上，它属于知识论、认识论的内容。其主要研究的内容从知识论角度看包括：知识的起源、知识的定义、知识的本质、知识的属性、知识的发展规律、知识的分类等。从认识论的角度看主要包括：人的认识活动是如何发生的，认识何以可能，我们能认识什么，真理的标准，主客体关系等。

在这些研究中，知识的分类问题是其中一个基础问题。任何一本关于知识组织、知识管理或者知识论的书籍对此都会有所提及。本文对分类及知识分类现象的本质、意义做了概述。作为我自身的学习过程，先占个位，后续可能会对元知识相关问题展开更多的记录和探讨。

知识现象与知识分类的困难

知识分类是人类“分类”思想在知识领域的应用。人类对事物的分类历史可以说和人类文明史一样长。没有分类思维，人类社会也不会发展到今天。分类本质上是对事物进行区分，是一种重要的认识活动。我们能将一只狗和一个苹果区分出来，本质上是因为它们属于不同的类别。所以，分类认识行为本身产生了知识。

在人类的进化过程中，首先是对实物进行分类，如对不同的动物、植物分类。这本身是人类求得生存的必要条件，这种对实物的分类相对容易理解。当我们的智人祖先不知道在什么时候什么原因，突然某一天调动了大脑的智慧余量来认识和改造世界后，整个人类的知识进入暴增模式。用《人类简史》作者赫拉利的观点来说，人类从此学会了虚构和讲故事。知识这个东西它不是一个实体，不是一个可以吃的苹果。在·赫拉利看来知识无疑是一种虚构的东西。但依靠这种虚构，人类社会发展到了今天。所以，千万不要以为我们今天有了互联网才说生活在虚拟世界，我们生活在虚拟世界已长达万年之久。按照如今的科技发展趋势，我们生活的虚拟程度和迷乱程度只会越来越高。

人类知识发展分化到今天，大大小小的学科分类统计起来已达上万种之多。知识之间错综复杂的关系使得对知识这种虚构性东西进行分类变得极为困难复杂。因为知识不同于实物，它是人类思维的产物，其分类问题就不如实物那般直观。我们一般容易把人类的知识比喻为知识的大树，有树根、树干、树枝。其实，我认为把人类知识看着一棵树，仅仅是因为这种树状结构很符合人类的认知习惯。实际情况分析来看，它已经不是树，而是一张知识的大网，是极为复杂的网状结构。要从这张大网之中，理出一点头绪其困难可想而知。

知识分类的现实意义

人类知识已经成为一张大网，而且还在不停的增长。那么对知识分类问题，是否是一个伪命题，是否有必要，对知识进行分类有何价值和意义。这个问题本身成为一个值得思考的问题。我认为，一个人再怎么厉害，都无法耗费一生的时间穷尽掌握人类的全部知识。那么，作为每个个体我们对知识进行分类、不停的学习各种知识的意义和正确方式应该是什么呢？什么知识最值得学习呢？

我觉得，人作为自然的产物，首先得求存。学习知识的行为和求存息息相关。所以学习的目的无非是求得自身的生存和发展。而对我们个人而言，知识分类的核心意义就在于：通过俯瞰人类知识之网，结合自身的兴趣禀赋特点，通过知识分类作为基础手段，从人类知识大网中构建一小块属于自己的知识版图用于自身的存在和发展之需要。我想有大部分人的学习行为可以归为此目的。但这里面有一小部分人，除了满足生存需要，把不断扩大自身知识版图甚或是扩大人类知识之网当成了毕生的追求和乐趣。结合前面分析，这一小部分人里面又可分为两类人，小部分中的大部分属于“知识应用工程师”，他们从人类知识大网中抽取部分知识构建自己的知识版图用于自己的生存和发展。他们是工程师、实践者、行动派，他们不断的改造世界。小部分中的小部分则属于人类知识的创造者，他们直接参与了人类知识大网的编织。他们是思想家、科学家、理论家，他们不断的认识世界。所以，不要说什么“思想的巨人，行动的矮子”这样的话，思想决定行动，行动反馈验证思想，两者虚实结合共同推动人类文明发展。完全属于分工合作而已，不存在谁瞧不起谁的问题。要我说，自从人类开始大规模调用智力后，对人类文明其支配作用的是思想家的思想，是思想成为了主宰。因为你的任何言行，事先必须在头脑中存在一个思想观念，是思想观念决定了你的行为。用王东岳老师的话说，我们不是活在事实之中，而是活在思想家铺设的思想通道之中。可见思想的力量是多么强大。

前面谈了一些关于知识、知识分类、思想、理论的重要性。至于什么知识最值得学的问题，我觉得无疑是元知识。通过元知识可以快速学会并内化具体的知识。对于那些以不停扩大自己知识版图为乐趣的人，如果要取得更好的成效，除了不断强化元知识学习，我觉得别无他法。元知识构成了知识之网的支撑结构，通过这个结构可以掌握知识的产生、发展、变化之规律。元知识也是参与到编织人类知识之大网创造活动中的必备条件。

对于元知识问题就不再展开了，我还没有能力对其进行系统的阐述，但我对上述提到的一些结论深信不疑。写作本文的目的也是一个我自己探索此问题的小小开端。下面对知识分类的意义总结几个要点。

  • 知识分类是科学知识确立的标识：所谓科学就是分科之学，自从科学从哲学中分化出来之后，科学的发展过程就是知识分类过程。每一门科学的确立都是知识分类的结果，近代科学的出现就是以知识分类体系的建立为标志的。

  • 知识分类有助于知识的秩序化管理：如前所述，对事物进行分类是人类认知世界的普遍心理活动。对事物进行分类是一种极为有效的管理方法。通过对知识进行分门别类的组织，知识呈现出易于管理和认识的秩序。追求秩序是科学家们奋斗的目标。

  • 知识分类有利于知识的选择与传递：人类的知识传递方式多种多样，比如学校、网络、电视、广播等。这些传播媒体有其独特的传递方式。对知识的分类有利于不同传播手段选择不同的知识进行传递。

  • 知识分类有助于知识的生产与积累：分工产生效率，知识分类也一样，将知识划分为不同的范围，不同的研究领域有不同的专门人员投入研究，有利于知识的生产。分类为知识的系统化与规范化发展提供了保障，知识分类就是一个为实现知识系统化与规范化并对它进行逻辑建构的过程。

  • 知识分类有助于知识创新：知识分类研究是知识创新的基础，任何一个新理论的构建，都需要对前人的研究结果进行充分的整理消化，而这个过程就是按照特定类别把分散在世界不同的地方和不同的载体之中的研究成果进行集中搜寻、认知的过程。如今知识发展呈现三个向度：分化、交叉与综合。分类是为了更好的综合，分类是在一团乱麻之中理出秩序，然后在这个秩序之上建立新的综合，这种交叉和综合是如今知识创新的主要方式。

总结

知识分类是元知识体系中的基础内容，对于普通个体而言，知识分类是知识组织与知识管理的基本方式。它有利于我们构建属于自己的知识体系，有了结构良好、稳定的个人知识体系，就能够快速高质量的学习掌握新知识，而一个人的学习能力无疑才是立足当下及未来社会的核心能力。对于从事知识创新、理论创新者而言，知识分类为他们提供了理论研究的良好基础平台。通过对前人构建的大量分类秩序的知识进行调研、分析、交叉、综合、改进、完善从而创造出新的知识体系。

笔者认为知识分类现象既是科学发展的原因又是科学发展的结果，但它不是科学发展的最终目的。如今高度分化、细化的学科之间由于缺乏沟通交流，已经阻碍了科学知识的良性发展。所以，如今的学科综合、迁移、跨界创新成为知识创新发展的新趋势。其中系统科学理论的出现就是代表，系统科学包括老三论：信息论、控制论、系统论；新三论：耗散结构理论、协同论、突变论。对这些系统科学理论有个基本了解有助于我们跳出还原分化思维模式，提高自身的系统化思维能力。

本文对元知识体系中的知识分类问题进行了概述，下一篇会对具体的知识分类方式进行汇总整理，掌握知识分类的多种维度对提升元知识思维能力又更深入了一步。
