生命质量公式之体积

发表于 2017-01-21

前言

我想作为生活在这世界上的每一个正常人，都希望在短暂的一生中尽可能提高生命的质量，有一个幸福的人生。但关于生命质量的定义及如何提高自己的生命质量一般人缺乏一个形象直观的认识，导致在具体行为方面缺乏指导意义。本文将生命质量类比为物理学中物体质量公式，以更直观更容易理解的方式来理解生命质量问题，对每个渴望提升自己的人如何在日常生活中正确行动有一定参考价值。全文分两篇：体积篇和密度篇，本篇是体积篇，重点阐述生命的长度、宽度和高度。

生命质量公式

以物理学求物体质量公式作类比，可以将生命视为一个立方体， 求立方体质量公式 = 体积 x 密度 = 长 x 宽 x 高 x 密度。相应的可以将生命质量公式定义为：生命质量 = 生命的长度 x 生命的宽度 x 生命的高度 x 生命的密度。 由此公式可知要提高生命的质量，需要尽可能提高四个量中的每一个量的值，下面针对每个量在人生中的含义进行类比说明。

生命的长度

生命的长度可以理解为生命的时间维度，就是说生命活得不能太短暂。追求长生不老是人类一直以来的梦想，如今据说有不少的高科技企业在研究如何让人类克服死亡以求得永生。这里不打算对这样疯狂的想法及可能实现之后产生的“吓尿式“影响展开想象和讨论，笔者仅就我们每个人拥有的常规肉身做一点提醒说明。

应该说作为一个普通的正常人，都希望能活得健康一点，长寿一点，但是如今高节奏的生活和工作，很多人以透支身体健康为代价去换取一些东西，这对当事人是否值得，其实是有必要思考的。本来一个成年人能为自己的所作所为负责，因此如果是处于自身意愿又没有危害他人的前提下，一个人如何对待自己的身体及健康本没有什么好评价的。问题出在很多人并没有在行事之前有深入的思考，并没有在深思熟虑的前提下做出主动选择，而是基于冲动基于随波逐流的方式作出并非明智的行为。结果是当危害发生时，才后悔莫及，才明白什么对自己是最重要的，可惜为时已晚。

关于身体健康问题，笔者不打算在具体的行为指导方面展开细说，主要也是因为我自身目前对此还没有什么科学的认识。我只想提醒大家：如果要提高自己生命的质量，身体健康这个维度显然是不容忽视的。作为我们一切行为的根本，我们的身体硬件是基础，需要我们对自身身体有一个科学的认识，并保持敏感和警觉，时刻注意关照好自己的身体。另一方面，我们学习、工作、成家立业的人生旅途中会伴随着我们身边的朋友和亲人的支持帮助，你不是一个孤立的个体，你至少要对某些人负有责任，从这个角度很容易明白一个道理：身体健康不仅仅是对自己负责同时也是对关心你及你关心的人负责。

关于如何保持身体健康的细节问题，虽然笔者目前对此还没有形成自己的科学方法，但也有一些零星的偏原则的思考要点可作为深入探讨的起点。注意：健康、医学话题过于严肃，责任重大，笔者并非这方面的专业人士，笔者这里所说的仅仅代表笔者自己在思维观念层面的当前认知。这些观念可能是错误的，请读者保持独立思考，若因为盲目相信了笔者的某些观点而给你造成了身体健康上的危害笔者概不负责 。下面是笔者的一些观点：

  • 人体是一个小宇宙，我们对此了解的其实知之甚少。不要对某些看似合理的大众行为盲目跟从，也不要过度迷信“专家”所说的话。

  • 身体有预警机制，不要忽视身体发出的信号。一个人对身体上的痛苦还有知觉，说明还不算太糟糕，说明身体预警感知机制还是正常的，趁这个时候就应该对身体及时关照调整。相反，一些人表面上挺好，可能还向别人夸口说看我自己啥事没有，其实这个时候此人的身体可能已经丧失了感知、知觉机制，实际情况可能已经非常糟糕。

  • 饮食和睡眠对健康影响巨大，这个大家都明白。只是很多人都难以做到平衡的饮食和规律的作息时间。关键是需要提高这方面的意识，并且要结合自身的特点以科学方法为指导。

  • 任何一件事情深入下去都成为一门高深的学问，医学问题、健康问题更是如此，作为普通人不可能要等到成为专家才去实施健康管理计划。不必去追求关于健康的绝对正确的真理，因为压根儿就不存在什么绝对真理。笔者觉得普通人要注意两点：一是你总得相信点什么；二是你不应把重点放在搞明白什么是正确的，而应该是去避免犯一些明显的常识性的错误。

  • 关于总得相信点什么，笔者认为我们还是得相信科学。从历史经验看，显然人类的寿命和健康是呈明显提高趋势的，而这得益于医疗科技的进步。笔者认为随着基因技术、智能信息技术在医疗领域的应用，不远的未来个性化医疗将成为普及，而这对我们每个人的健康是真正具有革命性的影响。关于相信科学，笔者要强调的是应该以科学精神的方式去相信科学，就是说对于健康、医疗相关的技术和信息你得有追根溯源的精神，得找到信息的源头并依据事实证据做出判断，而不是随便从某个渠道或者某个人那里获得一个信息，你就轻易相信了。以科学精神去相信某件事情，这事其实最难。

  • 一点感冒之类的小毛病可尽量扛一扛，给免疫系统工作表现的机会。不要一患上点小问题就着急上医院，免疫系统长期不工作，等于主动放弃自身拥有的强大武器。得了病进行治疗，不要一开始就采用激进手段，这同样可能破坏身体免疫系统，任何事情都要付出代价，应该对此有所权衡。不过，对于身处一个”求快“的当下社会现实，无论是患者还是医疗机构方似乎都挺难做到对彼此负责。笔者在这里只能无奈的说一句正确的废话：尽量少去医院。

  • 你持有的健康观念对你的最终健康起关键影响，这就是关于健康问题的价值观。比如你如何看待人体，如何看待生命科学，如何看待现代医学，如何看待被很多人吐槽的中医。还有一个更根本的问题：你如何看待”生死“。一个人持有的观念不容易被别人直接感知，但观念才是影响一个人行为方式的最根本因素。所以时常对自己持有的健康观念进行审视，应该会有更高的概率让自己保持健康。

生命的长度除了从肉身的健康和长寿角度考虑，还可以延伸到精神价值维度。一个人的肉身在短短几十年的时间里就会永久性消失，而一个人给世界留下的精神遗产却可以在时间的长河中永存。所以，把生命质点放到一个大尺度的时空坐标中进行审视，能够跨越时空，照耀人类不断前行的只能是哪些高贵生命所散发出的精神光辉。

生命的宽度

生命的宽度首先可以理解为生命的空间维度，在地理空间上可以在这个星球多走一走，多见识不同地方的人文风貌。这里很容易想到旅游的问题，从提高生命质量的角度看，旅游就应该变得更有价值，不要仅仅停留在看个热闹，看看风景的层面。

生命的宽度在个人知识和智慧层面延展开来，就是要提高自己的知识广度和见识。对常见的知识领域基本概念、原理有大致的了解，在和对应专业人士沟通时，能知道对方在说什么，在需要的时候能基于某个点进行深入的学习。

现代科学的学科分支越来越细，专业人员局限在某个细分领域相互之间缺少沟通协作。这一现象产生的不良影响已经被越来越多的有识之士深刻的意识到，如今大量的创新来自跨学科人才对不同知识领域的融合。所以，能跳出自己专业的细小夹缝，对人类整个知识大树有全局融会贯通的能力属于大师级人物，这个时代正在呼唤这样的大师。

关于知识的广度问题，在教育层面我理解就是“通识教育”。遗憾的是，通识教育是我们目前整个教育体系非常缺失的。要在这方面有所提高，在学校所受的教育对我们帮助不大（说严重点，帮助不但不大，甚至可能还有负面影响），更多的只能靠自己。好消息是，随着互联网的发展，整个世界的一体化进程越来越快，人与人之间相互的连接协作成本趋近于零。整个社会体系、科学体系呈现开放趋势，我相信会有越来越多内在动机驱使的人会涌现出来以自己过人的综合能力为他人为社会提供更多的价值。

生命的高度

生命的宽度是从水平空间视角来看待知识的广度，相应的，生命的高度是从垂直空间视角来看待知识的视野和深度。如果以地面作为参照，生命的高度可以向上延伸到高空，向下钻进地下深层。

向高空看，把视野放到整个宇宙，这会带来两个效果。一是由于视野空间变得极大，看到的整体格局就完全不一样，更容易看到问题的全貌和重点，以及解决问题的新思路。二是可以提升自己的胸怀，站在宇宙大尺度看，整个地球不过是宇宙中的一粒尘埃。再在时间尺度想一想整个宇宙138亿年的漫长历史，在无限的时空尺度下，人类作为地球尘埃中的一个物种是多么渺小，而你我作为芸芸众生的一员更是不值一提，完全可以忽略。这会产生一个什么样的心灵冲击呢，就是你会感觉你当前遇到的困难、焦虑、痛苦、悲伤、心中正在打的小算盘变得毫无必要毫无意义。你开始怀疑自己已经拥有的或者苦苦追求的究竟有何意义。此时难道你不应该问自己三个哲学问题吗：我是谁？我从哪儿来？我要到哪儿去？当我们开始追问人生价值和人生意义的时候，我想我们的生命质量已经开始提升。

向地下看，就是指的专业深度。从当今社会的发展趋势看，分工产生效率这个命题依然成立。自己参与到一个分工中去，在专业领域精耕细作，成为细分领域的权威是每个普通人立足现代社会比较容易达成的目标。在这方面还要有一定的未来眼光，就是我们当前的大部分职业工种在未来很快都会淘汰消失，而未来急需的大量新职业分工现在有些刚有苗头，有些还没有命名。这个时候就应该面向未来做准备，而不是抱残守缺。要有这样的未来判断，前面提到的知识广度和视野就非常重要。

关于专业深度，我再强调一下：这是每个普通人的真正机会所在，现今大量个体的崛起就是明证。一切外部障碍已经消失，剩下的仅仅是一个动机、选择、努力与方法的问题。而这个问题是可以通过刻意练习和有效学习来解决的，最终有数量可观的普通人都可以达成自己的人生小成就。

本篇结语

用物理质量公式类比生命质量公式，本篇重点对体积部分进行了类比阐述。长、宽、高分别对应生命的身体健康、知识广度、知识视野及专业深度。其实每一个方面展开来讨论都是很大的话题，基于笔者有限的阅历和当前认的知只能说一些看似有一定道理的鸡汤。好在关于鸡汤问题笔者专门撰文进行了思考提出了自己的一点看法，在笔者心中对鸡汤文字没有不屑和贬义，这样也就释然了。可参考：关于鸡汤的鸡汤：读写鸡汤文章的正确姿势。

关于健康、医学方面的问题非常严肃，我在文中也专门提醒了，我说的观点仅代表我个人当前的认知，可能是不正确的或不合时宜的。如有不当的地方，请专业人士批评指正。

下篇我会接着说说生命的“密度”问题，生命密度这个维度是更重要的一个维度。也是我们一般人理解的人生价值更容易体现的方面。
